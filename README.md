### Prerequisites
- Node.js (version 18 or newer)

### Installation & Running

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd breezit-angular
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the application**
   ```bash
   npm start
   ```

4. **Open in browser**
   ```
   http://localhost:4200
   ```
